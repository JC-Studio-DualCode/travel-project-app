## Description

**CityVerse** is a React Single Page Application that allows users to explore countries and cities, view detailed information, manage points of interest, and add reviews.

The project focuses on clean UI, intuitive navigation, and real-time data persistence using Firebase.  
It was developed as **Project 2** during the Ironhack Web Development Bootcamp to practice React, routing, CRUD operations, and component-based architecture.

## Features

- Browse countries and cities
- View detailed city information
- Add and edit cities
- Manage points of interest with images
- Add and delete reviews
- Responsive and modern UI
- Real-time data storage with Firebase Realtime Database

## Tech Stack

- React
- React Router
- Axios
- Firebase Realtime Database
- CSS Modules
- Vite
